    Для выполнения домашнего задания были созданы отдельные классы.
    
    Задание 1.
    В классе ListOfNamesEditor реализован метод namesByOddIndexes(List<String> names),
который принимает на вход список имен и возвращает строку в виде, соответствующем условию 
задачи.

    Задание 2.
    В классе ListOfNamesSorter реализован метод sortNamesByUpperCase(List<String> names),
который принимает на вход список имен и возвращает список строк в верхнем регистре 
и отсортированные по убыванию (от Z до А).

    Задание 3.
    В классе NumbersFromArray реализован метод numbersEditor(String[] badNumbers),
который принимает на вход массив, элементы которого представляют собой строки, состоящие
из нескольких цифр, разделенных запятой. Метод возвращает строку, содержащую все эти цифры
в отсортированном виде, разделенные запятой.

    Задание 4.
    В классе InfinityStream реализован метод infinityStreamGenerator(long seed, long a, long c, long m),
который принимает на вход параметры "линейного конгруэнтного генератора" и возвращает
бесконечный стрим. Корректность вводимых параметров проверяется в методе 
isValidParameters (long seed, long a, long c, long m). В случае, если один из вводимых
параметров не будет соответствовать условию, генерируется RuntimeException.

    Задание 5.
    В классе StreamsZipper реализован статический метод zip(Stream<T> first, Stream<T> second),
который "перемешивает" два потока, которые приходят в качестве аргументов, и возвращает 
результирующий поток.
